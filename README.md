# Stores-Annual-Sales-Analysis
Annual Sales data analysis of a store


>>Project Overview

-This project analyses sales performance data of Vrinda Store using Microsoft Excel to uncover business insights, customer trends, and revenue patterns. The objective was to transform raw transactional data into actionable insights using Excel-based data analytics tools.

>>Objectives

-Analyse overall sales performance and revenue trends.

-Identify top-performing states, categories, and channels.

-Understand customer demographics (age, gender).

-Track order status distribution and fulfillment efficiency.

-Create an interactive dashboard for business decision-making

>> Tools & Techniques Used

-Microsoft Excel, Pivot Tables, Pivot Charts, Data Cleaning & Preprocessing, Slicers for Interactivity, Basic Data Modelling

>>Dataset Description

The dataset includes:
Order ID, Order Date, Customer Gender & Age Group, State, Sales Amount, Order Status, Sales Channel, Product Category

>>Key Insights

-Identified highest revenue-generating states contributing significantly to total sales.

-Women customers contributed a larger share of total revenue.

-Majority of orders were successfully delivered, indicating strong fulfillment rate.

>> Dashboard Features

-Monthly Sales Trend Analysis

-Sales by Gender & Age Group

-State-wise Sales Distribution

-Order Status Breakdown

-Channel-wise Sales Contribution

-Interactive filters using slicers

>> Business Recommendations

-Target high-contributing customer demographics with personalized ads.

-Strengthen underperforming sales channels.

-Optimize inventory planning for peak months.

>> Learning Outcomes

-Strengthened Excel data analytics skills.

-Improved business insight generation from raw data.

-Built interactive dashboards for stakeholder reporting.

-Practiced structured data storytelling.
